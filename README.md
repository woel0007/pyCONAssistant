# pyCONAssistant

This module provides additional reporting capabilities to the ICON CMO accounting software.

To use this module, create a file called user.conf with the following contents:  
```
phone=[church phone number]  
user=[username]  
pass=[password]  
```

For example:  
```
phone=5555555555  
user=JoePUser  
pass=password  
```
